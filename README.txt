CV-PROJECT
This project is based on https://github.com/angular/angular-seed.git
Its goal was to discover the basics of angularjs.
To load the project on your PC and start it in command line :
cd <place you want to put the project>
git clone https://github.com/paul-larauze/cv-project.git <folder name>
cd <folder name>  
npm start

Then in the browser : http://localhost:8000/app/#/presentation
